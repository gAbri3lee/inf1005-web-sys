## Project Overview
This project is a responsive, accessible, and data-driven web application developed for the **INF1005 – Web Systems and Technologies Group Project**. The website is built for a **fictitious company/organization** and is designed to demonstrate both front-end and back-end web development skills.

According to the project guide, the system must include:
- a main landing page,
- a common navigation menu,
- sub-pages for products and/or services,
- an “About Us” style page,
- and back-end functionality that supports creating, storing, and updating data.

## Objectives
The objective of this project is to apply the technologies and concepts taught in INF1005 by building a fully functioning web application that is:
- responsive across devices,
- accessible,
- secure,
- data-driven,
- and supported by full CRUD database functionality.

## Key Features
This project is designed to meet the requirements stated in the module guide.

### Front-End Features
- Responsive landing page introducing the company/organization
- Shared navigation menu across all pages
- Product and/or service pages
- About Us page
- Mobile-friendly interface using **HTML5**, **Bootstrap**, and **CSS**
- Dynamic client-side functionality using **custom JavaScript**

### Back-End Features
- Server-side functionality using **PHP**
- Database integration using **MySQL**
- Full **CRUD operations** for selected system data
- Secure form handling with validation and sanitization
- Protection against common web vulnerabilities such as:
  - Cross-site scripting (XSS)
  - SQL injection
- Secure password protection for user accounts
